# poo-to-do

Este proyecto es un todo, proyecto ejemplo para programación

## Estructura de carpetas

```bash
.poo-to-do/
todo_app/
│
├── app/
│   ├── models/
│   │   └── task.py
│   ├── services/
│   │   └── task_service.py
│   ├── storage/
│   │   └── file_storage.py
│   └── ui/
│       └── ui_cli.py
├── data/
│   └── tasks.json
│── requirements.txt
├── README.md
├── .gitignore
└── main.py
```

## Instalación

1. Crear en un entorno virtual

```bash
python -m venv .venv
```

2. Activar el entorno virtual

```bash
# Windows
.venv\Scripts\activate
# Linux/MacOS
source .venv/bin/activate
```
